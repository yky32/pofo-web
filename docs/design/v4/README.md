# Design Version 4 — Soft Atelier

**Status:** Exploring (`design/v4-explore`)  
**Branch:** `design/v4-explore`

Warm stone surfaces, muted rose accent, soft rounded cards — editorial and approachable.

## Mood

- Soft atelier / lifestyle brand
- Warm gray-stone background (not pure cream, not pure white)
- Muted rose CTAs and hearts
- Rounded everything (pills, 2xl cards, soft photo corners)
- Friendly premium, not stark museum or dark cinema

## vs previous

| | V1 Studio Paper | V3 White Cube | **V4 Soft Atelier** |
|--|-----------------|---------------|---------------------|
| Base | Warm cream | Cool pure white | **Warm stone + blush wash** |
| Accent | Stone ink | Black | **Muted rose** |
| Type | Cormorant + DM Sans | Space Grotesk + Manrope | **Fraunces + Plus Jakarta** |
| Radius | Soft / mixed | Square 0 | **Fully rounded** |
| Feel | Print studio | Museum | **Soft lifestyle atelier** |

## Tokens

- Background: warm stone with soft rose/amber radial wash
- Rose accent for primary CTAs
- Cards: white-ish paper, soft shadow, `rounded-2xl`
- Photos: `rounded-2xl` + gentle lift shadow

## Type

- Headings: Fraunces (soft editorial serif)
- UI: Plus Jakarta Sans

## Patterns

- Soft hero with staggered rounded frames
- Feature cards in panels
- Client gallery: rounded cover bottom, rose hearts
- Dashboard banner with warm gradient overlay
