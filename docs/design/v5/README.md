# Design Version 5 — Contact Lab

**Status:** Exploring (`design/v5-explore`)  
**Branch:** `design/v5-explore`

Cool slate surfaces, steel-blue accent, contact-sheet language — pro tool clarity for photographers.

## Mood

- Photographer workstation / modern SaaS
- Cool blue-gray (not cream, not pure white museum, not dark, not rose)
- Mono labels, frame numbers, progress bars
- Precise, calm, efficient

## vs previous

| | V1 | V3 | V4 | **V5 Contact Lab** |
|--|----|----|----|--------------------|
| Base | Warm cream | Pure white | Warm stone + blush | **Cool slate** |
| Accent | Ink | Black | Rose | **Steel blue** |
| Type | Serif editorial | Grotesk geometric | Soft serif | **Geist (system modern)** |
| Shape | Mixed | Square 0 | Full soft round | **Medium radius (tool)** |
| Feel | Print studio | Museum | Lifestyle | **Pro lab / product** |

## Tokens

- Background: cool slate-tinted white
- Steel blue CTAs and active states
- Panels: light card + subtle border
- Photos: slight radius, ring border
- Meta: mono uppercase micro labels

## Patterns

- Contact sheet strip with frame numbers
- Gallery cards with selection progress bar
- Client view: dense equal grid (true contact sheet)
- Job metadata bars (`JOB · FRAMES · PRIVATE`)
